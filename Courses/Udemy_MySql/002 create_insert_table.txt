========================Insert into table=======================================

After getting into a specific database, if we need to insert value into a table then
following thing we need to do
1.single row insertioon :-)

	INSERT INTO <table_name>(all column_name with comma(,) )
	VALUES(values_according to column with comma(,) );
eg.
	INSERT INTO cats(name, age)  OR INSERT INTO cats(age, name)
	VALUES("Blue steele",5);        VALUES(3,"beth);

here order with column don't matter, but it must consistent with values order.
	INCORRECT eg.---> INSERT INTO(name, age)    OR     INSERT INTO(age, name)
				VALUES(3, "beth");         VALUES("beth", 4);
The just above command show error,as its column not consistent with values order

2. Multiple Line insertion :-)

	INSERT INTO <table_name>(and name of each column seperated by comma(,) )
	VLAUES (row values consisted with  above order of column name seperated by (,) ),.,.;
eg.
	INSERT INTO cats (name, age) 
	VALUES ('Meatball', 5), 

  		 ('Turkey', 1), 
  		 ('Potato Face', 15);	


	NOTE --:	 Always try to wite string or text in single quote(') as other dbms expect mysql don't use double qoute.
			 and we can use escape char to print single or double quote(') as element value. same as in python with single and double quote.

=====================================Default============================================

At time of creation of table we can assign default value for each column if user did not enter the vlaue then
default value would set for that inerstion.

	eg. CREATE TABLE cats
	    (
		name VARCHAR(34) DEFAULT 'unamed',
		age INT(2) DEFAULT '99'
		);



CODE: Adding DEFAULT Values
Define a table with a DEFAULT name specified:

CREATE TABLE cats3  (    
    name VARCHAR(20) DEFAULT 'no name provided',    
    age INT DEFAULT 99  
);
Notice the change when you describe the table:

	DESC cats3;

Insert a cat without a name:

	INSERT INTO cats3(age) VALUES(13);

Or a nameless, ageless cat:

	INSERT INTO cats3() VALUES();

Combine NOT NULL and DEFAULT:

	eg. CREATE TABLE cats4  
		(    
    		name VARCHAR(20) NOT NULL DEFAULT 'unnamed',    
    		age INT NOT NULL DEFAULT 99 
		);


==================================PRIMARY KEY============================================

-- One way of specifying a PRIMARY KEY

	CREATE TABLE unique_cats (
		cat_id INT PRIMARY KEY,
    		name VARCHAR(100) NOT NULL,
    		age INT NOT NULL
	);


-- Another option:


	CREATE TABLE unique_cats2 (
		cat_id INT,
    		name VARCHAR(100) NOT NULL,
    		age INT NOT NULL,
    		PRIMARY KEY (cat_id)
	);


===============================AUTO_INCREMENT==========================================

--  AUTO_INCREMENT( we don't need to give now any value it will assign a value itself and take care of uniqueness if it's a keys.


CREATE TABLE unique_cats3 (
    cat_id INT AUTO_INCREMENT,
